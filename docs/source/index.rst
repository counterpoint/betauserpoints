Welcome to BetaUserPoints' documentation!
===================================

**BetaUserPoints** is a Joomla 4 component that tracks points for users. Points can be earned by user actions and spent on services or products.
It offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
